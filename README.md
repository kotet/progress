Easy progress reporting for D inspired by [Python's one with the same name](https://pypi.python.org/pypi/progress).  
There are 6 progress bar, 4 spinner, 4 counter, and... Dman.

## Progress bar 
 - Bar
 - ChargingBar
 - FillingSquaresBar
 - FillingCirclesBar
 - IncrementalBat
 - ShadyBar

## Spinner
 - Spinner
 - PieSpinner
 - MoonSpinner
 - LineSpinner

## Counter
 - Counter
 - Countdown
 - Stack
 - Pie

## Dman
 - DmanSpinner
